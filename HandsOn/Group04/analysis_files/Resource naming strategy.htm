Defining the resource naming strategy :

 

We are going to use the slash for resources pattern path because of the amount of data generated by sensors and different captures and the hash # for ontological terms paths.

 

The domain of the generated Dataset is:

o http://www.smart-city.GP04-LDATA.org/

 

Ontological terms path:

o   http://www.smart-city.GP04-LDATA.org/city/ontology/Road-traffic-accidents#

 

Ontological terms pattern:

o   http://www.smart-city.GP04-LDATA.org/city/ontology/Road-traffic-accidents#<term_name>

Example:

o    http://www.smart-city.GP04-LDATA.org/Leeds/ontology/Road-traffic-accidents#hasRefNumber

o   http://www.smart-city.GP04-LDATA.org/Leeds/ontology/Road-traffic-accidents#hasNumberOfVehicles

 

Individuals path:

o   http://www.smart-city.GP04-LDATA.org/city/resource/

 

Individuals pattern:

o   http://www.smart-city.GP04-LDATA.org/city/resource/<resource_type>/<resource_name>

Example:

o   http://www.smart-city.GP04-LDATA.org/Leeds/resource/Date/AccidentDate

o   http:/www.smart-city.GP04-LDATA.org/Leeds/resource/string/WeatherConditions
